# DoseCerto Trab de Laboratorio de Software II - Prof Luiz 
# Rodrigo, Amilton, Kevyn e Fabricio

O DoseCerto é um web app utilitário mobile-first voltado para a área da saúde. Seu objetivo é realizar a conversão matemática exata de dosagens de medicamentos injetáveis, da dose prescrita (mg) para o volume a ser extraído (mL e UI).
